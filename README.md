# Rick and Morty App

A simple web application that consume the [The Rick and Morty API](https://rickandmortyapi.com/) and present the list of characters and them details.

## Technologies

Project made with React (Typescript), using:

- Redux
- Redux Toolkit
- Material-UI
- Hooks
- Axios
- React router dom
